# AAL90 Lookup Tables

This folder contains AAL90 lookup tables used to define region names and ordering
for TVBToolkit AAL90 workflows.

## Files

- `custom_lookuptable_AAL.txt`: AAL90 interleaved left/right ordering.
- `symmetric_lookuptable.txt`: AAL90 hemisphere-grouped symmetric ordering.

These lookup tables include MRtrix-style lookup metadata and licensing comments
in the file headers. They are small atlas metadata files only; they do not
contain participant data.
